Nmap
####

The tool Nmap is supported by PTP thanks to the following classes.

.. warning::

    The development of the Nmap ranking system has been postponed to after the
    GSoC.
    For now, the classes below only parse the XML reports generated by Nmap but
    do not rank the discoveries.
