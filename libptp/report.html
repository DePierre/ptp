Basic Reports¶

synopsis:Define the basic report classes that will represent the data from the report file(s).

AbstractReport¶

class libptp.report.AbstractReport[source]¶

Abstract representation of a report provided by a pentesting tool.

Note

This class will be extended for each pentesting tool. That way, each tool will add its own reporting specificities.
